# Computer Vision for Attendance Recognition: Bibliometric Analysis

This repository contains the data and scripts used in the article:

"Computer Vision to Attendance Recognition: A Bibliometric Analysis"

Contents:

* `data/scopus.csv`: metadata exported from Scopus.
* `data/wos.txt`: metadata exported from Web of Science.
* `data/merged_dataset.csv`: consolidated dataset after duplicate removal.
* `bibliometric_analysis.R`: R script used to process data, perform bibliometric analyses, and generate figures.

Software requirements:

* R (version 4.x or higher)
* Bibliometrix package

The repository is provided to support transparency, reproducibility, and future research in the area of computer vision applied to attendance monitoring systems.
